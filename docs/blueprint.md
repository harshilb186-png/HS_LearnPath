# **App Name**: LearnPath Pro

## Core Features:

- Curriculum Mapping: Allow users to map a curriculum to a specific career path using data ingested from the linked GitHub repo.
- Content Summarization: Use AI to summarize key learning points from various modules and content from the Github repo. Display key summaries of learning for the user.
- Skill Assessment Tool: Integrate an AI-powered tool that assesses a user's current skill level in relation to their desired career path and suggest areas of improvement.
- Progress Tracking: Implement a progress tracking system that monitors user advancement through the curriculum and highlights completed vs. remaining tasks.

## Style Guidelines:

- Primary color: Dark blue (#3F51B5) to convey trust, knowledge and stability, in alignment with educational resources.
- Background color: Light gray (#EEEEEE) for a clean and professional look.
- Accent color: Teal (#009688) for interactive elements and progress indicators, creating an accessible and user-friendly experience.
- Body text: 'Inter' (sans-serif) for a modern, neutral, easily-readable style. Headlines: 'Space Grotesk' (sans-serif) for short amounts of impactful text
- Use flat design icons related to learning, skill development, and progress tracking.
- Clean, card-based layout with clear separation of content sections. Prioritize readability and easy navigation.
- Subtle transition animations for smooth user experience when navigating between curriculum modules and skill assessments.
